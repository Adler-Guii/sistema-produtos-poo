# 💼 Sistema de Produtos - POO

Projeto desenvolvido em Java para praticar conceitos fundamentais de Programação Orientada a Objetos.

## 🧠 Conceitos aplicados

- Herança
- Polimorfismo
- Sobrescrição de métodos (@Override)
- Uso da palavra-chave `super`

## ⚙️ Regras de negócio

- Produto comum: retorna o preço original
- Produto com desconto: aplica desconto ao preço
- Produto com taxa: adiciona taxa ao preço

## 🚀 Tecnologias

- Java
- Eclipse

## 📌 Objetivo

Reforçar conceitos de POO através de exercícios práticos, focando em reutilização de código e organização de classes.
